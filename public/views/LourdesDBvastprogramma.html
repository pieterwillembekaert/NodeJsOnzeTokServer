Database vast programma

Op deze pagina worden alle vast activiteit aangemaakt. Deze activiteit worden gekoppeld aan de weides
Onderaan de pagina is een formulier terug te vinden.

  • id: niet invullen, dit is informatief
  • wat: naam van de activiteit
  • waar: locatie
  • duur: hoelang de activiteit start
  • start: tijdstip wanneer de activiteit start
  • stop: tijdstip wanneer de activiteit stopt
  • benodigdheden
  • beschrijving: korte omschrijving van de activiteit




Opmerking: als de data bij het inladen niet onmiddellijk zichtbaar is refresh de pagina.

download: klik op de knop download om de huidige database te downloaden. Dit bestand kan dienen als backup.
